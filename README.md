# 🎨 С чистого листа

Учебный проект по верстке адаптивного веб-сайта.  
Проект представляет собой информационную статью о способах борьбы со страхом «чистого листа» при написании кода или текстов.

---

## 📌 1. О проекте

Одностраничный сайт, разработанный для отработки навыков **HTML** и **CSS**.  
Дизайн выполнен в темной теме с яркими акцентами.

---

## 🛠 2. Технологии

В проекте используются:

1. **HTML5**
2. **CSS3**
3. Адаптивная верстка
4. Flexbox
5. Кастомные шрифты:
   - *Inter*
   - *EB Garamond*

## 🧩 4. Особенности верстки

### 4.1 Семантическая структура

Использованы семантические теги HTML5:

- `<header>` — шапка сайта с навигацией  
- `<main>` — основной контент  
- `<section>` — логические блоки страницы  
- `<footer>` — подвал с контактами и копирайтом  

---

### 4.2 Ключевые блоки страницы

1. **Шапка сайта** — логотип и навигационное меню  
2. **Обложка** — крупный заголовок с нестандартным выравниванием  
3. **Интро** — краткое введение и дата публикации  
4. **Причины страха** — список с объяснениями  
5. **Способы решения** — карточки с советами  
6. **Подвал** — копирайт и ссылки на социальные сети  

---

## 🎨 5. Стилизация

### 5.1 Цветовая схема

- Основной фон — `#1e1e1e` (темная тема)  
- Акцентный цвет — `#FEE42A` (желтый для ярких блоков)  

### 5.2 Дизайн-решения

- Чередование темных и ярких секций  
- Карточки со скругленными углами (`border-radius: 30px`)  
- Использование фото авторов  
- Контрастная типографика  

---

## 📱 6. Адаптивность

1. Максимальная ширина контента — **360px** (мобильная версия)  
2. Резиновая верстка с использованием:
   - `max-width`
   - процентных значений  
3. Гибкое расположение элементов на **Flexbox**
